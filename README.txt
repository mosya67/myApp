приложение можно запустить через консоль, заранее перейдя в папку проекта.
приложение использует базу данных PostgreSql, строка поделючения к ней изменяется через код в файле ProjectConstants
сама строка выглядит так: "Host=localhost;Port=5432;Database=mydb;Username=postgres;Password=12345"

myapp 1
создает таблицу в базе данных

myapp 2 ФИО дата_рождения пол
создает запись в таблицу(ФИО записывать через пробелы)
пол указывается одним из ниже перечисленных вариантов
мужской: m, M, male, Male
женский: f, F, female, Female
пример создания записи:
myapp 2 Иванов Александр Матвеевич 21-05-2000 M

myapp 3
выводит все записи в базе данных

myapp 4
создает 1.000.000 рандомных записей и еще 100, где ФИО начинается с 'F' и пол мужской

myapp 5
возвращает записи, у которых ФИО начинаетс с 'F' и пол мужской

myapp 6
демонстрирует изменения во времени выполнения запроса на выборку(пункт 5).
